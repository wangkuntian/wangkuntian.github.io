Dockerе…Ҙй—ЁпјҲдә”пјү

Sep 20, 2018

д»“еә“пјҲRepositoryпјүжҳҜйӣҶдёӯеӯҳж”ҫй•ңеғҸзҡ„ең°ж–№гҖӮ

жіЁеҶҢжңҚеҠЎеҷЁжҳҜеӯҳж”ҫд»“еә“зҡ„е…·дҪ“жңҚеҠЎеҷЁпјҢжҜҸдёӘжңҚеҠЎеҷЁдёҠеҸҜд»ҘжңүеӨҡдёӘд»“еә“пјҢиҖҢжҜҸдёӘд»“еә“дёӢжңүеӨҡдёӘй•ңеғҸгҖӮеҜ№дәҺд»“еә“ең°еқҖdl.dockerpool.com/ubuntuжқҘиҜҙпјҢdl.dockerpool.comжҳҜжіЁеҶҢжңҚеҠЎеҷЁең°еқҖпјҢubuntuжҳҜд»“еә“еҗҚгҖӮ

Docker Hub

е®ҳзҪ‘

еҲӣе»әе’ҢдҪҝз”Ёз§Ғжңүд»“еә“

дҪҝз”Ёregistryй•ңеғҸеҲӣе»әз§Ғжңүд»“еә“

еҸҜд»ҘдҪҝз”Ёе®ҳж–№жҸҗдҫӣзҡ„registryй•ңеғҸжқҘжҗӯе»әжң¬ең°з§Ғжңүд»“еә“зҺҜеўғгҖӮ

жӯӨж—¶жң¬ең°е°ҶеҗҜеҠЁдёҖдёӘз§Ғжңүд»“еә“жңҚеҠЎпјҢзӣ‘еҗ¬з«ҜеҸЈдёә5000гҖӮ

й»ҳи®Өжғ…еҶөдёӢпјҢдјҡе°Ҷд»“еә“еҲӣе»әеңЁе®№еҷЁзҡ„/tmp/registryдёӢгҖӮ
еҸҜд»ҘйҖҡиҝҮ-vеҸӮж•°жқҘе°Ҷй•ңеғҸж–Үд»¶еӯҳж”ҫеңЁжң¬ең°зҡ„жҢҮе®ҡи·Ҝеҫ„дёҠгҖӮ

з®ЎзҗҶз§Ғжңүд»“еә“й•ңеғҸ

дҪҝз”Ёdocker tagе°Ҷubuntuй•ңеғҸж Үи®°дёә10.211.55.11:5000/testпјҲж јејҸдёәdocker tag SOURCE_IMAGE[:TAG] TARGET_IMAGE[:TAG]пјү

10.211.55.11жҳҜжҲ‘зҡ„IPең°еқҖгҖӮ

дҪҝз”Ёdocker pushдёҠдј ж Үи®°зҡ„й•ңеғҸгҖӮ

зҺ°еңЁеҸҜд»ҘеңЁд»»ж„ҸиҝһжҺҘеҲ°10.211.55.11зҡ„жңәеҷЁдёӢиҪҪиҝҷдёӘй•ңеғҸгҖӮ

еҮәзҺ°й”ҷиҜҜ
