Ubuntu Server那点儿事

Dec 11, 2018

目录

  1. 修改屏幕分辨率
  2. 修改字体
  3. 设置时区并更新时间
  4. root密码重置
  5. root ssh登录

修改屏幕分辨率

  1. 打开grub文件（$ vim /etc/defalut/grub），并修改参数GRUB_CMDLINE_LINUX的值。参数值参考如下。
640x480 800x600 1024x768 1280x1024
256 0x301 0x303 0x305 0x307
32k 0x310 0x313 0x316 0x319
64k 0x310 0x314 0x317 0x31A
16M 0x312 0x315 0x318 0x31B

  1. $sudo update-grub

  2. $sudo reboot

修改字体

  1. $sudo dpkg-reconfigure console-setup

设置时区并更新时间

  1. $sudo tzselect
    选择Aisa，China。

  2. $sudo cp /usr/share/zoneinfo/Asia/Shanghai /etc/localtime

  3. $sudo apt install ntdate

  4. $sudo ntdate time.windows.com

root密码重置

  1. $sudo passwd

root ssh登录

  1. 修改/etc/ssh/sshd-config文件中的PermitRootLogin no为yes
