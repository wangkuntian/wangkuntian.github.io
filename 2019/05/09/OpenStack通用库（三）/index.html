OpenStackйҖҡз”Ёеә“пјҲдёүпјү

May 09, 2019

oslo.dbжҳҜй’ҲеҜ№SQLAlchemyи®ҝй—®зҡ„жҠҪиұЎгҖӮ

зӣ®еҪ•

  1. дҪҝз”ЁSQLAlchemyзҡ„sessionе’Ңconnection
  2. дҪҝз”ЁOpenStackдёӯйҖҡз”Ёзҡ„SQLAlchemy modelзұ»

дҪҝз”ЁSQLAlchemyзҡ„sessionе’Ңconnection

oslo.dbжҸҗдҫӣдәҶoslo_db.sqlachemy.enginefacadeжЁЎеқ—жқҘиҺ·еҸ–sessionе’ҢconnectionпјҢжңүдёӨз§Қж–№жі•жқҘдҪҝз”ЁenginefacadeпјҢеҚіеҮҪж•°иЈ…йҘ°еҷЁпјҲdecoratorпјүе’ҢдёҠдёӢж–Үз®ЎзҗҶеҷЁпјҲcontext mangagerпјүгҖӮиҝҷдёӨз§Қи°ғз”Ёж–№ејҸйғҪйңҖиҰҒжҸҗдҫӣдёҖдёӘдёҠдёӢж–ҮеҜ№иұЎгҖӮдёҠдёӢж–ҮеҜ№иұЎеҸҜд»ҘжҳҜд»»дҪ•Pythonзұ»гҖӮиҝҷж ·еҒҡзҡ„зӣ®зҡ„жҳҜжҸҗдҫӣдёҖдёӘз»ҹдёҖи§„иҢғзҡ„sessionдҪҝз”ЁжЁЎејҸпјҢйҒҝе…Қи°ғз”ЁиҖ…дҪҝз”ЁдёҚеҪ“йҖ жҲҗж•°жҚ®еә“дәӢеҠЎпјҲtransactionпјүзҡ„ж»Ҙз”Ёе’ҢеөҢеҘ—гҖӮ

еҪ“дҪҝз”ЁиЈ…йҘ°еҷЁжЁЎејҸзҡ„ж—¶еҖҷйңҖиҰҒеҜ№contextеҜ№иұЎеҒҡзү№ж®ҠеӨ„зҗҶпјҢи°ғз”Ёtransaction_context_providerиЈ…йҘ°contextеҜ№иұЎгҖӮ

з®ЎзҗҶе‘ҳеҸҜд»ҘйҖҡиҝҮй…ҚзҪ®ж–Үд»¶жқҘй…ҚзҪ®oslo.dbзҡ„и®ёеӨҡйҖүйЎ№пјҢжҜ”еҰӮпјҡ

з”ЁжҲ·еҸҜд»ҘеңЁдҪҝз”Ёж•°жҚ®еә“д№ӢеүҚи°ғз”Ёoslo_db.sqlalchemy.enginefacade.configure()ж–№жі•жқҘж”№еҸҳе·Іжңүзҡ„й…ҚзҪ®гҖӮ

еёёз”Ёй…ҚзҪ®йҖүйЎ№гҖӮ

й…ҚзҪ®йЎ№ = й»ҳи®ӨеҖј зұ»еһӢ иҜҙжҳҺ
backend = sqlalchemy еӯ—з¬ҰдёІ еҗҺеҸ°ж•°жҚ®еә“ж ҮиҜҶ
connection = None еӯ—з¬ҰдёІ sqlalchemyз”ЁжӯӨжқҘиҝһжҺҘж•°жҚ®еә“
connection_debug = 0 ж•ҙеһӢ sqlalchmeyзҡ„debugзӯүзә§пјҢ
0иЎЁзӨәдёҚиҫ“еҮәд»»дҪ•и°ғиҜ•дҝЎжҒҜпјҢ
100иЎЁзӨәиҫ“еҮәжүҖжңүи°ғиҜ•дҝЎжҒҜ
connection_trace = False еёғе°”еһӢ жҳҜеҗҰжҠҠPythonзҡ„и°ғз”Ёж ҲдҝЎжҒҜеҠ еҲ°SQLзҡ„жіЁйҮҠдёӯ
db_inc_retry_interval = True еёғе°”еһӢ иҝһжҺҘйҮҚиҜ•ж—¶пјҢжҳҜеҗҰеўһеҠ йҮҚиҜ•д№Ӣй—ҙзҡ„ж—¶й—ҙй—ҙйҡ”
db_max_retries = 20 ж•ҙеһӢ иҝһжҺҘйҮҚиҜ•зҡ„жңҖеӨҡж¬Ўж•°пјҲ-1иЎЁзӨәдёҖзӣҙйҮҚиҜ•пјү
db_max_retry_interval = 10 ж•ҙеһӢ иҝһжҺҘйҮҚиҜ•ж—¶й—ҙй—ҙйҡ”пјҢеҚ•дҪҚдёәз§’
idle_timeout = 3600 ж•ҙеһӢ иҝһжҺҘиў«еӣһж”¶д№ӢеүҚзҡ„з©әй—Іж—¶й—ҙпјҢеҚ•дҪҚдёәз§’
max_overflow = None ж•ҙеһӢ еҰӮжһңи®ҫзҪ®дәҶпјҢиҝҷдёӘеҸӮж•°дјҡиў«зӣҙжҺҘдј з»ҷsqlalchemy
max_pool_size = None ж•ҙеһӢ еңЁдёҖдёӘиҝһжҺҘжұ дёӯпјҢжңҖеӨ§еҸҜеҗҢж—¶жү“ејҖзҡ„иҝһжҺҘж•°
max_retries = 10 ж•ҙеһӢ жү“ејҖиҝһжҺҘж—¶жңҖеӨ§йҮҚиҜ•ж¬Ўж•°пјҲ-1иЎЁзӨәдёҖзӣҙйҮҚиҜ•пјү
retry_interval = 10 ж•ҙеһӢ жү“ејҖиҝһжҺҘйҮҚиҜ•ж—¶зҡ„ж—¶й—ҙй—ҙйҡ”

дҪҝз”ЁOpenStackдёӯйҖҡз”Ёзҡ„SQLAlchemy modelзұ»

oslol_db.sqlalchemy.modelsе®ҡд№үдәҶдёӨз§ҚMixinпјҡTimestampMixinе’ҢSoftDeleteMixinгҖӮдҪҝз”ЁTimestampMixinж—¶SQLAlchemy modelдёӯдјҡеӨҡеҮәдёӨеҲ—create_atе’Ңupdate_atпјҢеҲҶеҲ«иЎЁзӨәи®°еҪ•зҡ„еҲӣе»әж—¶й—ҙе’ҢдёҠдёҖж¬Ўдҝ®ж”№ж—¶й—ҙгҖӮ

SoftDeleteMixinж”ҜжҢҒдҪҝз”Ёsoft deleteеҠҹиғҪгҖӮ

дёҚеҗҢDBеҗҺз«Ҝзҡ„ж”ҜжҢҒгҖӮ

дёҚеҗҢbackendе…·дҪ“е®һзҺ°ж—¶пјҢйңҖиҰҒе®ҡд№үеҰӮдёӢеҮҪж•°иҝ”еӣһе…·дҪ“DB APIзҡ„е®һзҺ°зұ»гҖӮ
